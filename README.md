# ThriveWell AI 🌙
### *Your mind deserves to thrive.*

ThriveWell AI is a premium mental wellness platform designed to provide students and individuals with a safe, judgmental-free space to heal, reflect, and grow. Combining empathetic AI companionship with expert human care.

## 🚀 Features
- **Empathetic AI Chat**: 24/7 support with real-time streaming and crisis detection.
- **Expert Counselor Network**: Connect with verified mental health professionals.
- **Mood Tracking**: Visual emotional history using interactive Recharts.
- **Daily Journaling**: A rich-text reflective space for personal growth.
- **Emergency Resources**: Quick access to local and global helplines.

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS 4, Glassmorphism, Premium UI Utilities
- **AI**: Vercel AI SDK, OpenAI GPT-3.5/4
- **Database**: Supabase, Prisma ORM
- **Auth**: NextAuth.js

## 📦 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Setup Environment**:
   Duplicate `.env.example` to `.env` and fill in:
   - `OPENAI_API_KEY`
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `NEXTAUTH_SECRET`
4. **Run Development Server**:
   ```bash
   npm run dev
   ```

## 🛡️ Privacy & Security
All conversations are encrypted. Users can choose to remain 100% anonymous. We prioritize your privacy above everything.
